Memory with Tang

Posted on 2017-03-27 Edited on 2024-05-26 In Prose
Symbols count in article: 495

心中幻想过的最大的温暖
是唐老师握着我的手帮我擦干血迹 帮我用创口贴贴好伤口
而不骂我一句

细细数来 已经有很多感动
温暖至今 随意挑拣一件 便可暖我入眠
从不担心因不设防而受伤 亦师亦父的安心

Chapter 1
那天尽管冠着错误的姓称呼你 全办公室却没说一个字
“她如果成绩好可以让她去一班。”
“让她自己选，她想去哪个班。”转过头，“一班是实验班。”
“你看，她自己选的一班。”


Chapter 2
“我们现在一轮复习已经复习完了。” 尽管后来知道你只是吓唬
“我这里正好多了一套生物教辅。”
“你挨着挨着做。如果做不来就问同学。”
我扭头看你。“或者问我也可以。”


Chapter 3
第二天周一 升旗仪式
下到操场
你只是简单问问
“他们有没有笑你？”


Chapter 4
才到一班不久 在等电梯时碰到你
“哪一科最难？” “生物。”
并不完全时没有意识到你不习惯开玩笑 当时只是有点任性吧
“你看你也不来找我问题。”


Chapter 5
才来不久 全班收生物作业
“交了作业的都是乖娃娃。”
发下来，第一本是我的
“你们连新同学的姓都不知道啊？”


Chapter 6
终于在一个下午鼓起勇气找他问题
问完
“是不是生病了？” 扯着嘴角笑着点头
“看你精神不是很好。” 笑容升一级 走出办公室
