Context

本來還在考慮要不要接著昨天的寫，後來想想罷了罷了，重頭開始吧。今天晚上打算先複習課件然後看題目，找quizlet再看看。

仔細想了一下，還是昨天那樣“記錄問題-解釋問題”的方式更合適。再想了想，好傢伙我大概還是看不完吧。看到哪裡算哪裡。

好傢伙，又開始走神。想到明天10:30就要考試，我只能繼續找能讓自己集中注意力在複習上的辦法——最後找到一個，把YouTube上的學習視頻放在屏幕的右上角。其實用手機說不定也可以的。（說到這兒我就想要雙屏了。）真真是個好辦法：同時有計時，有白噪音，還有屏幕上的變動。不過說到屏幕上的“變動”，或者“放置”，真的好有點ADHD喔。

想起來，應該給自己的博客加一個全文搜索。考完再做吧。

  1. Lecture 1
    • ARM:32bit地址長度
    • word的長度取決的processor，一般來說是32個bits
    • program counter每一次增加4
  2. Lecture 2
    • reverse subtraction：RSB rz, ry, rx means subtract the value in yyy from the value in xxx and place it in zzz.
    • Multiplication： If the result is more than 32 bits long the destination register, rz, only holds the bottom 32 bits of the result and the rest is lost.
    • MLA: Multiply and accumulate. MLA rz,ry,rx,rw: rz = ry * rx + rw
    • BIC rz, ry, rx: Bit Clear, means “ry AND NOT rx”
    • LDR
    • automatic update: The automatic updating is identified by the !, ‘pling’.
  3. Lecture 3
    • SUB or SUBS: they perform the same function except SUBS will set or clear the flags and SUB will not change the flags from their current setting.
    • MOVCS r12, #114: means move 114 into r12 if the carry flag is set.
    • NaN： all ones for the number part of float
  4. Lecture 4
    • stack pointer: r13
  5. Lecture 5
    • UNREAD
