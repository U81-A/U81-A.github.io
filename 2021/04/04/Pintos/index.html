Pintos

Posted on 2021-04-04 Edited on 2021-04-05 In Log
Symbols count in article: 2.5k

昨晚失眠。今早一覺起來，就知道Pintos寫不完。晚上一時想起所謂的恐懼與憤怒，抑鬱與麻木。種種前緣。

準備按照番茄🍅時鐘來嘗試一次，看看效率能否變高。Tomato Todo是早就下載在了手機上，今天進去一看，自習室要求綁定手機號，遂告別自習室。


The 1st 25min

剪了指甲。Habitica的週期性提醒真的很有用。

在斯坦福的官網上面有這麼一段：

Before you read the description of this project, you should read all of the following sections: 1. Introduction, C. Coding Standards, E. Debugging Tools, and F. Development Tools. You should at least skim the material from A.1 Loading through A.5 Memory Allocation, especially A.3 Synchronization. To complete this project you will also need to read B. 4.4BSD Scheduler.

所以決定從這裡開始。只是存個檔，因為GDB還沒看懂——但是有不少文章說做這個project GDB很重要。

想起來似乎應該在Ubuntu裡面裝一個VS Code。先把分辨率調整好了，然後調整了顯存和內存的大小，似乎變得順暢一點了。但是斯坦福的“Coding Standards”我還是沒有看懂。


The 2nd 25min

首先裝了VS code，也算是Dual Task了。按照VScode的教程來裝。下一步就是等待C/C++的擴展下載安裝、VScode建立文件樹。順便想辦法在Mac這邊也打開了同樣的內容（希望是一樣的，畢竟不知道make到底會對文件幹什麼）。也就是說瀏覽代碼確實需要在本機上，但是修改代碼在Ubuntu VM的VScode上面會快很多：因為VIM用不熟。考慮到可能有不少複製粘貼代碼的內容，於是決定在Ubuntu上下載一個QQ。

在安裝的時候繼續看之前要求的前置信息，Loading: It’s not important to understand exactly how the loader works. 覺得這個有必要於是存個檔。

LUNCH

and little nap

其實我覺得我應該去睡一覺，但是沒有睡。而且也沒有打遊戲。但是效率並不怎麼高，同時還提醒自己不要隨便亂喝咖啡。前幾天一喝咖啡就心跳加速、這次來月經疼得厲害，無時無刻不在提醒我不要亂造。所以打算吃完晚飯再去睡一會兒，然後熬夜把代碼寫完。

看不進去，打算洗頭。以及覺得不知道今晚幾點能睡。

於是來到了第二天中午——我實在是個菜雞。

抄了別人的bullet list如下：

  • 重写timer_sleep()，避免busy waiting

    • thread_mlfqs为false时使用优先级调度，为true时使用多级反馈队列调度

  • 实现优先级调度

    • schedule应选择高优先级线程
    • 高优先级线程就绪时，应抢占
    • 锁、信号量、条件变量应优先唤醒高优先级线程
    • 当前线程降低自身优先级后，应yield
    • 需考虑优先级反转，实现优先级捐赠（仅对于锁）
    • 线程不会修改其它线程的优先级

  • 实现多级反馈队列

    • 此时优先级仅由公式决定，不受其它因素影响
    • 实现定点数运算
    • 实现优先级的更新

然後開始copy代碼。

開始給自己建立一個短暫的routine：關於如何從博客園看完全看不懂的思路、去GitHub上看完全找不到在哪裡的代碼、以及在自己卡成PPT的ubuntu VM上找到對應的代碼做修改這件事情：

  1. 按照“代碼復現”部分確定一段需要修改的代碼。
  2. 按照博文中的提示在clone到本地的倉庫裡（用VScode），通過全文搜索找到應該修改、添加的地方。
  3. 通過上下文以及全文搜索在ubuntu VM裡找到自己的代碼沒修改的地方。
  4. 對照macOS和ubuntu VM修改代碼。
  5. 找到下一步需要修改的代碼，重複步驟一。

一直有一個大問題：不知道某一項任務要花多久。而這在計系似乎是大忌。

被OS按在地上錘。


第二天的第一次🍅⏰

總之就是先只給自己三個小時，然後看一看效果如何。這25分鐘就先看報告有些什麼要求、老師給的tutorial，以及預估自己寫報告要多久。在表示“不敢喝咖啡”之後，其實已經不知道碳酸飲料和檸檬茶到底誰對我更有效了。以及我覺得現在並沒有上學期期末那段時間那麼嚴重的焦慮，所以也實在沒打算考慮咖啡。

嚴格來說知道report要寫什麼只花了9分鐘，但是確定要從哪裡去copy報告還是花了一點時間。然後還剩兩個半小時。


然後shutdown了⋯⋯


重啟中。


⋯⋯


縫縫補補。

咖啡過期了，他媽的。
